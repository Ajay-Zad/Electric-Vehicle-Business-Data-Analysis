Electric Vehicle Business Data Analysis

Electric Vehicle Business Data Analysis Dashboard Link: https://public.tableau.com/views/EletricVehicleDataAnalysis/Dashboard1?:language=en-US&publish=yes&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

Data Sources: The dashboard integrates data from a single table (.csv file) to provide a holistic view of the dairy business operations.

This project analyzes electric vehicle data using Tableau. The dataset includes the following key columns:
1) VIN (1-10): Vehicle Identification Number
2) County, City, State, Postal Code: Location information
3) Model Year, Make, Model: Vehicle details
4) Electric Vehicle Type, CAFV Eligibility: Vehicle classification and alternative fuel eligibility
5) Electric Range: Distance covered on a full charge
6) Base MSRP: Manufacturer's Suggested Retail Price
7) Legislative District, DOL Vehicle ID: Legal and registration info
8) Vehicle Location, Electric Utility, 2020 Census Tract: Geographic and utility data

Tools Used
Tableau for data visualization and insights.

The dashboard is designed to help users analyze business performance, identify trends, and make data-driven decisions to optimize operations and sales strategies within the Electric Vehicle Business.

NOTE: The original CSV file contained various irregularities, including NULL values, incorrect data, improperly formatted data, and empty cells, all of which were addressed during the creation of the dashboard using Tableau tool
